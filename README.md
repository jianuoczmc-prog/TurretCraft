# TurretCraft (Forge 1.20.1)

A lightweight turret mod for Minecraft Forge 1.20.1.
Place turrets, load ammo, and protect your base with smooth aiming and firing animations.

## Latest Update
- **v1.1.1**: Fixed turret dismantle behavior (level is preserved and materials/upgrades are refunded correctly).

- Release docs index: [`说明文档/README.md`](./说明文档/README.md)

## Features (EN)
- **Multiple turrets** for base defense (more coming).
- **Turret GUI**: check ammo, change target mode, unload ammo.
- **Smart targeting**: automatically tracks valid targets.
- **Spherical range**: intuitive detection radius.
- **Smooth aim & fire animations**: turret yaw rotation, barrel pitch, recoil feedback.
- **Simple survival-friendly** gameplay loop: craft → place → load → defend.

## Requirements (EN)
- Minecraft **1.20.1**
- Forge **47.x**
- Java **17**

## Installation (EN)
1. Install Minecraft Forge **1.20.1 (47.x)**.
2. Put `TurretCraft-1.20.1-*.jar` into:
   - Windows: `%AppData%\.minecraft\mods`
   - macOS: `~/Library/Application Support/minecraft/mods`
   - Linux: `~/.minecraft/mods`
3. Launch the game using the **Forge** profile.

## How to Use (EN)
1. Craft a turret and place it in the world.
2. Open the turret GUI to **load ammo** and set the **target mode**.
3. Watch it automatically track and fire at targets within range.

## Known Issues / FAQ (EN)
- If the mod doesn’t show up, make sure you launched the **Forge** profile and restarted the game after adding the jar.
- Conflicts can happen with other mods that heavily change entity targeting/AI.
