# CurseForge — Changelog for v1.1.0 (from v1.0.0)

## English

### Highlights

- Turret Upgrade System (levels + upgrade slots + Upgrade action).
- New items: Turret Firmware (x2 recipe) + Turret Manual (Book + Firmware).
- GUI now shows Level/Damage/Cooldown/Ammo Save; new tooltips + new GUI textures/icons.
- Added Traditional Chinese (zh_tw).
- Projectile entity spawning improved via NetworkHooks (multiplayer).
