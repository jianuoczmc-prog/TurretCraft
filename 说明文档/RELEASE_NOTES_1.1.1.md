# TurretCraft v1.1.1 — Release Notes

## English

### Fixes

- Fixed an issue where turret **level data was lost** after dismantling.
- Fixed an issue where dismantling a turret **did not refund upgrade/material items** correctly.
- Dismantling now properly preserves progression state and returns expected materials.

### Notes

- This is a **bugfix patch** for v1.1.0.
- No recipe or balancing changes in this version.

---

## 简体中文

### 修复

- 修复了炮台在**拆除后等级丢失**的问题。
- 修复了炮台拆除时**升级/材料未正常返还**的问题。
- 现在拆除炮台会正确保留进度数据，并返还应返还的材料。

### 说明

- 这是针对 v1.1.0 的**问题修复补丁**。
- 本版本未调整配方与数值平衡。
